# Deck Contract

A manifest has `artifact_id`/`deck_id`, `artifact_type: slide_deck`, `audience: student`, positive `version`, `brief_id`, `grade_band`, optional `cefr`, `target_language`, `instruction_language`, non-empty `objective_ids`, `methodology_lineage`, `source_references`, `authority`, `accessibility`, `brand`, versioned `dependencies`, `slides`, and `teacher_notes`. Each student slide has a stable `slide_id`, `surface: student`, objective IDs, one main purpose, and ordered content blocks. Interactions state teacher cue, learner action, timing, and reveal behavior. Media blocks require factual source context, `alt_text`, and a non-media `fallback_text`. Teacher notes map exactly once to every slide ID. Print fallbacks preserve the learning action without relying on animation, sound, or color.
