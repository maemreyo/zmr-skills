# Learning Science Protocols

Apply these protocols when designing sequences, lessons, explanations, practice, or reteaching.

## Retrieval and spacing

- Name the objective retrieved by each prompt.
- Schedule retrieval after initial teaching and again after increasing delays.
- Mix current and previously taught objectives; do not make every review block predictable.
- Record the next review point in a learning sequence or progress record.

## Worked examples and fading

- Model a complete example before independent performance when the task is novel or complex.
- Ask the learner to explain the decisive clue or step.
- Fade support from complete example to completion problem to independent response.
- Restore the smallest missing scaffold when evidence shows overload.

## Cognitive load

- Remove decorative or duplicate information that competes with the objective.
- Chunk instructions and keep necessary representations spatially close.
- Introduce one new source of complexity at a time.
- Use modality only when it reduces processing demand or serves the language objective.

## Transfer

- Start with a near-transfer task that changes surface details.
- Add a far-transfer task that changes context while preserving the target principle.
- Require explanation or evidence so correct guessing is not mistaken for transfer.

## Metacognition

- Make success criteria visible before work begins.
- Ask learners to predict, monitor, and explain rather than merely rate confidence.
- End with a concrete next action linked to evidence.
