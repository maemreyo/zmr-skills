# Shared Teaching Brief

Capture only values supported by the request, an approved artifact, or a labelled assumption.

- Learner: grade band, age band, optional independently supplied CEFR, class size, ClassProfile ID or approved Learner Context Snapshot ID, accessibility and support needs.
- Intent: topic, English domain, desired outcome, stable objective IDs, success evidence, prerequisites, misconceptions, duration, delivery context.
- Language: target language, instruction language, bilingual balance.
- Pedagogy: methodology, rationale, pinned status, curriculum/exam alignment, differentiation.
- Scope: lesson/unit/sequence, sessions, artifact types, audiences, counts, forms, length constraints.
- Authority: required/preferred/reference sources, answer authority, rubric authority, media/transcript authority.
- Constraints: must include, avoid, prohibited assumptions, privacy, difficulty, accessibility, print, technology.
- Delivery: requested formats, student/teacher surfaces, colour/grayscale, print requirements.
- Identity: `zamery`, `rooted in strength`, and supplied assets only.

Use provenance values `explicit`, `inferred`, `defaulted`, `unresolved`, and `not_supplied` for optional CEFR. Keep grade and CEFR independent. Full StudentCards never enter the Teaching Brief.

Authority precedence is safety/privacy/legal/answer separation; current teacher instruction; required teacher sources; approved brief/blueprint/sequence; preferred/reference sources; then labelled inference/default. Material conflicts require a teacher decision.

When review helps, show `Known`, `Inferred`, and one `Needs your decision` item.
