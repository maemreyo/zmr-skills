# Methodology Registry

Choose the smallest useful method. Never default to inverse thinking merely because it is available.

| Method | Primary use | Do not use when | Required structure | Outputs |
|---|---|---|---|---|
| inverse thinking | Strong misconception boundary | No meaningful wrong model exists | disaster, clues, safe zone, rule, transfer | contrast, board rule, probes |
| semantic anchoring | Vocabulary meaning and retrieval | The issue is syntax, not meaning | core meaning, selection clues, collocations, memorable example | vocabulary board |
| contrastive pairs | Easily confused forms or sounds | Contrasts overload the learner | matched examples, changed feature, decision rule | comparison table |
| active recall | Consolidation after initial teaching | Learners have no initial model | cue, retrieval pause, reveal, correction, delayed return | oral or written prompts |
| why-wrong reasoning | Distractor or error diagnosis | Public error analysis could shame a learner | candidate, evidence, rejection, correct rule | teacher-led reasoning |
| concept mapping | Relationships among several ideas | One linear distinction is enough | nodes, labelled links, examples | map plus explanation |
| roleplay | Pragmatics and speaking choices | Learners lack language support or safety | low-pressure rehearsal, role cues, retry | dialogue and speaking moves |
| film-based learning | Listening, inference, pragmatics | Clip rights/access or transcript is unavailable | scene context, transcript clues, prediction, evidence | clip-independent transcript task |
| timed practice | Fluency with already understood content | Speed would replace understanding or create pressure | age-appropriate round, accuracy check, reflection | short rehearsal |

Timed work is optional and age-appropriate. Roleplay always permits rehearsal before performance. Film-based work must remain usable from a supplied or teacher-authored transcript without embedded media.
