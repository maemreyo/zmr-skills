# Practice Progression

1. Worked example: show the target thinking, not only the final response.
2. Guided: supply prompts, cues, or partial structure with immediate checking.
3. Independent: remove the key scaffold while preserving the same objective.
4. Retrieval: revisit after spacing or an intervening task without the model visible.
5. Transfer: apply the same objective in a meaningfully new context.

Guided and independent practice are always required. Omit a worked example only when the approved lesson already contains one. Omit retrieval only when the artifact is a single immediate classroom segment. Omit transfer only when the teacher explicitly requests controlled rehearsal or safety requires mastery first; label the omission.
