# Exercise catalog

Question variety is multidimensional. Select one or more values from each axis before choosing a classroom pattern.

| Axis | Examples |
|---|---|
| Interaction | choice, matching, ordering, cloze, editing, labelling, oral response, essay |
| Response mode | selected, constructed, spoken, performance, mixed |
| Cognitive operation | remember, understand, apply, analyze, evaluate, create |
| Evidence | word, sentence, passage, image, dialogue, audio, video, authentic source |
| Scoring | exact key, accepted set, partial credit, analytic rubric, observation |
| Context | independent item, shared stimulus, timed section, pair work, home viewing |

## Classroom patterns

| ID | Pattern | Strongest use | Common failure |
|---|---|---|---|
| P01 | single choice | discrimination/retrieval | option clues |
| P02 | multiple response | combining evidence | unclear partial credit |

| P26 | essay | sustained argument | prompt tests background knowledge |

| P29 | read-aloud | decoding/prosody | no observation rubric |
| P30 | information-gap pair task | speaking/listening | unequal roles |
| P31 | roleplay | pragmatics/fluency | no rehearsal/support |
| P32 | oral retell | comprehension/fluency | memory load dominates |
| P33 | video timestamp response | listening/noticing | ungrounded transcript |
| P34 | flashcard retrieval | spaced recall | recognition only |
| P35 | sorting/manipulative | K–2 concepts | dense written directions |
| P36 | portfolio evidence | authentic transfer | unclear rubric |

For K–2, favor oral, pointing, sorting, matching, drawing, and teacher-scribed responses. When listening practice lacks audio capability, provide a teacher script and clearly labeled TTS-ready text; never claim an audio file exists.
