# Video learning sequence

| Phase | Purpose | Examples |
|---|---|---|
| Before | activate schema and set a listening purpose | predict from title, preview essential terms, state one focus question |
| During | collect evidence without constant interruption | gist check, ordered details, note completion, inference, language noticing |
| After | consolidate and transfer | summary, discussion, speaking retell, writing response, IELTS-style listening transfer |

Use fewer interruptions for short clips. Place questions after an evidence window, not before the required information. Source-dependent answers cite segments; open personal responses cite the learning objective instead.

For home viewing, include device-light instructions and a printable alternative. For classroom use, include teacher pause points and expected discussion time.
