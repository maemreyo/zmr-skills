# Accessibility and publishing

- Record whether captions exist and whether they are accurate enough for the task.
- Provide a transcript or structured text alternative when listening is not the assessed construct.
- Avoid color-only cues and tiny hotspot targets.
- Make instructions usable without autoplay.
- Test keyboard navigation, focus, contrast, mobile layout, media playback, and caption behavior on the target platform.
- The H5P exporter creates a host-resolved content package. Import it only into a platform that already provides compatible declared libraries; validate after import.
- Prefer the LMS quiz engine for secure summative testing. H5P is strongest for formative interaction and learning analytics.
