# YouTube and transcript policy

A public watch URL is not a transcript license. YouTube Data API caption download requires authorization scopes and access to the caption track. Do not scrape or represent automatically retrieved text as authorized merely because a video is public.

Safe modes:

1. ask the teacher to paste/upload the transcript;
2. use captions obtained through an authorized channel workflow;
3. use licensed text supplied in scope;
4. create non-factual observation prompts from the public link and mark them for teacher verification.

Preserve the original URL, video ID, transcript authority, and teacher-verification status in the manifest.
