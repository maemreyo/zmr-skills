# Question taxonomy

Model variety with independent axes instead of one flat “question type” list.

| Axis | Examples |
|---|---|
| Language domain | vocabulary, grammar, pronunciation, reading, writing, listening, speaking, language use |
| Interaction | single/multiple choice, true/false, matching, ordering, categorization, cloze, transformation, error correction, table/note/summary completion, labeling, dictation, essay, oral response, timestamped video response |
| Response mode | selected, constructed, spoken, performance, mixed |
| Cognitive operation | remember, understand, apply, analyze, evaluate, create |
| Evidence | isolated sentence, passage, image, audio, video, dialogue, authentic source, portfolio |
| Scoring | exact key, accepted set, partial credit, analytic rubric, holistic rubric, teacher observation |
| Context | independent item, shared stimulus set, timed section, collaborative task, home viewing |

Use a balanced matrix. A 100-item bank with ten surface formats can still be monotonous if every item only recalls a fact. Variety must change evidence and thinking, not decoration alone.
